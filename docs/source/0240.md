### 二叉搜索树

   * 二叉搜索树又称二叉排序树（中序遍历可以得到有序数列），是二叉树的衍生结构，可以实现快速查找数据。

   * 特点：对于任意节点，其左子树的值小于节点值，右子树的值大于节点值。

### 代码实现

    class TreeNode:
        def __init__(self, value) -> None:
        self.value = value
        self.left = None
        self.right = None

    class BST:
        def __init__(self) -> None:
            self.root = None

### 基本操作

   * 搜索某个节点：从根节点开始与目标值进行比较，若当前节点值小于目标节点，则证明待搜索节点位于当前节点右侧，递归搜索右子树，反之则递归搜索左子树。

    # 查找指定节点
    def Search(self, value):
        current = self.root
        while current:
            # 查找到对应值
            if current.value == value:
                return current
            # 当前节点值小于待查找节点，尝试搜索右子树
            elif current.value < value:
                current = current.right
            # 搜索左子树
            else:
                current = current.left
        return None
        
   * 插入节点：类似于搜索操作，但区别在于插入操作需要判断待插入位置是否为空，若不为空则需继续递归搜索。

    # 插入节点
    def Insert(self, value):
        if not self.root:
            self.root = TreeNode(value)
            return
        current = self.root
        while current:
            # 当前节点值小于待插入节点，尝试向右子树插入
            if current.value < value:
                if not current.right:
                    current.right = TreeNode(value)
                else:
                    current = current.right
            if current.value > value:
                if not current.left:
                    current.left = TreeNode(value)
                else:
                    current = current.left
        
   * 删除节点：同样需要先进行搜索，在找到待删除节点后需要进行不同情况的处理。

### 小结

   1. 二叉搜索树是一种用于快速查找的二叉树
   2. 中序遍历二叉搜索树能够得到有序数列，因此二叉搜索树又叫二叉排序树
   3. 二叉搜索树的搜索性能不稳定，在极端情况下会退化成链表
