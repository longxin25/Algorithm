### 希尔排序

   * 希尔排序在插入排序的基础上，对待排序数据进行切割，对每个部分分别进行插入排序，待整体趋于有序后，对整体再进行插入排序。

---

### 代码实现

    def shell_sort(data):
        length = len(data)
        if length <= 1:
            return
        # 增量因子
        gap = 1
        while gap < length / 3:
            gap = gap * 3 + 1
        while gap > 0:
            # 对子序列进行插入排序
            for i in range(gap, length):
                value = data[i]
                j = i - gap
                while j >= 0 and data[j] > value:
                    data[j+gap] = data[j]
                    j -= gap
                data[j+gap] = value
            # 增量递减
            gap = gap // 3

### 性能分析

   * 时间复杂度：O(n)
   * 空间复杂度：O(1)

---

### 关键点

   1. 子序列处理方式和插入排序逻辑一致（1 改成 gap）
