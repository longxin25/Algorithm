### 插入排序

   * 插入排序是一种原地排序，思想为将待排序数据分为有序区间和无序区间，通过从无序区间依次提取元素插入到有序区间的合理位置，使得有序区间仍保持有序，最终达到整体有序。

   * 一般起始设置有序区间为 a[0] （第一个元素），需要进行 n - 1 趟插入操作。

---

### 代码实现

    def insert_sort(data):
        length = len(data)
        if length <= 1:
            return
        # 起始已排序区间：data[0],未排序区间：data[1...n]
        for i in range(1, length):
            # 待插入元素
            value = data[i]
            # 已排序区间末尾
            j = i - 1
            # 从已排序区间末尾向前遍历，将比待插入元素大的数全部向后搬移
            while j >= 0 and data[j] > value:
                data[j+1] = data[j]
                j -= 1
            # 搬移结束，插入元素
            data[j+1] = value
        
---

### 性能分析

   * 时间复杂度：插入排序一共需要进行 n - 1 次插入操作，每一次插入操作需耗费 O(n)，因此整体时间复杂度为 O(n²)。

   * 空间复杂度：插入排序是原地排序，虽然涉及数据搬移但没有借助额外空间，因此整体空间复杂度为 O(1)。

---

### 关键点

   1. 在搬移之前必须用局部变量记录下待插入元素，否则进行搬移会导致待插入元素的值丢失。
   2. 插入位置下标是当前遍历位置的下一位。
